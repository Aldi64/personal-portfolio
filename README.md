# Personal Portfolio

A personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **React** – UI framework
- **TypeScript** – Type safety
- **Tailwind CSS v4** – Utility-first styling
- **Vite** – Build tool

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is ready to deploy on **GitHub Pages**, **Vercel**, or **Netlify**.
